# 🌿 COCDECH — Coco D'Or Bio

Site vitrine Next.js 14 pour COCDECH, PME ivoirienne de valorisation des déchets de noix de coco.

## 🚀 Démarrage rapide

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## 🛠 Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**

## 📁 Structure

```
src/
├── app/           # Pages (App Router)
├── components/    # Navbar, Footer, UI
├── data/          # Données statiques
└── lib/           # Utils
```

## 📦 Déploiement

```bash
npm run build
```

Déployer sur **Vercel** (gratuit) : `vercel deploy`

## ✏️ Personnalisation

- Modifier les données dans `src/data/`
- Remplacer les placeholders d'images dans `public/images/`
- Mettre à jour les liens réseaux dans `src/data/social.ts`
- Adapter les textes directement dans les pages
